# Diplom_2: API тесты для Stellar Burgers

## Описание проекта
Автоматизированные тесты API для сервиса [Stellar Burgers](https://stellarburgers.education-services.ru).

## Тестируемые эндпоинты
1. **Создание пользователя**
   - Создание уникального пользователя
   - Создание уже зарегистрированного пользователя
   - Создание пользователя без обязательных полей

2. **Логин пользователя**
   - Вход под существующим пользователем
   - Вход с неверными данными

3. **Создание заказа**
   - С авторизацией и без авторизации
   - С ингредиентами и без ингредиентов
   - С неверным хешем ингредиентов

4. **Получение заказов пользователя**
   - Авторизованный пользователь
   - Неавторизованный пользователь
